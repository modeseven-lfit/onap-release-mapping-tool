Changelog
=========

This project follows `Semantic Versioning <https://semver.org/>`_ and publishes
release notes on the GitHub
`Releases <https://github.com/modeseven-lfit/onap-release-mapping-tool/releases>`_
page.

.. note::

   This project is in early development. The API and CLI interface may change
   between minor versions until ``1.0.0`` is released.
